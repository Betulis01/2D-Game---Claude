# TalentPanel

**File:** `core/src/main/java/com/Betulis/Game2D/game/ui/panels/TalentPanel.java`
**Extends:** `UIPanel`
**Role:** 3×3 grid of talent slots (non-draggable SpellSlots) for future talent tree functionality.

## Layout
- Position: `(screenW - 320, screenH/2 - 200)`
- Size: 164×184 (or PNG size after `loadPanelBg`)
- Background: `ui/panels/talent_panel.png` (generated by UIAssetGenerator)
- 3 columns × 3 rows, slot size 40px, gap 8px, 10px padding

## Constants
- `COLS = 3`, `ROWS = 3`, `SLOT_SIZE = 40f`, `SLOT_GAP = 8f`

## Dependencies
- `UIPanel`, `SpellSlot`

## Rules
- Slots are non-draggable (`draggable = false` passed to SpellSlot constructor).
