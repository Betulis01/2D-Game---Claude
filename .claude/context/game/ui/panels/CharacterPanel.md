# CharacterPanel

**File:** `core/src/main/java/com/Betulis/Game2D/game/ui/panels/CharacterPanel.java`
**Extends:** `UIPanel`
**Role:** Equipment panel displaying all gear slots (head, shoulders, chest, etc.) as SpellSlot widgets.

## Layout
- Position: `(20, screenH/2 - 250)`
- Size: 200×480 (or PNG size after `loadPanelBg`)
- Background: `ui/panels/character_panel.png` (generated by UIAssetGenerator)
- Left column (5 slots): Head, Shoulder, Back, Chest, Wrist
- Right column (5 slots): Hands, Waist, Legs, Feet, Relic
- Bottom row (4 slots): Ring×2, Trinket×2
- Weapon row (2 slots): Weapon left + right

## Key Methods
- `addEquipSlot(key, x, y, textures, fallback)` — internal helper; looks up texture by key name, creates SpellSlot child

## Dependencies
- `UIPanel`, `SpellSlot`
- `Map<String, Texture>` keyed by `equip_*` names (from UIManager's `equipTextures`)

## Rules
- Slot positions use hardcoded offsets from `x`/`y` and panel `w`/`h` constants — not affected by `loadPanelBg` w/h update since they're computed before the call.
