# SpellBookPanel

**File:** `core/src/main/java/com/Betulis/Game2D/game/ui/panels/SpellBookPanel.java`
**Extends:** `UIPanel`
**Role:** Displays available spells as draggable SpellSlot widgets. Opens via PanelMenuBar toggle.

## Layout
- Position: `(screenW/2 - 150, screenH/2 - 200)`
- Size: 300×200 (or PNG size after `loadPanelBg`)
- Background: `ui/panels/spellbook_panel.png` (generated by UIAssetGenerator)
- Fireball slot at `(x+20, y+100)` — draggable
- Lightning slot at `(x+120, y+100)` — draggable
- Labels drawn below each slot

## Fields
| Field | Type | Purpose |
|-------|------|---------|
| `spellSlots` | `List<SpellSlot>` | Exposed for UIManager drag wiring |

## Key Methods
- `getSpellSlots()` — returns list for UIManager to attach drag listeners

## Dependencies
- `UIPanel`, `SpellSlot`, `UILabel`, `SpellDefinition`

## Rules
- Spell drag is wired externally by UIManager; SpellBookPanel only holds the slots.
